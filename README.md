# [QuizifyReact](https://quizify-react.onrender.com)

This project is a React quiz app that allows users to answer questions within a specified time limit and view their results. It demonstrates the usage of the `useReducer` hook in managing state within a complex application.For preview [QuizifyReact](https://quizify-react.onrender.com)

## Features

- **Quiz Interface:** Users can start the quiz, answer questions, and see their progress.
- **Timer Functionality:** A timer counts down as users answer questions. When time is up, the quiz ends automatically.
- **Scoring System:** Users receive feedback on their performance and see their final score at the end.
- **Responsive Design:** The app is designed to work seamlessly across different screen sizes.

To get started with QizifyReact, follow these steps:

1. Clone the repository: git clone https://github.com/enesuraz/quizify-react.git
2. Navigate to the project directory: cd quizify-react
3. Install dependencies: npm install
4. Navigate App.jsx and remove comment between "For development" and add comment "For Production"
5. Open another terminal and type "npm run server" for starting json-server
6. Get into main terminal and type "npm run dev"
7. Open your browser and visit website that shown in terminal to view the website.
